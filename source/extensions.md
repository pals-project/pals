# Extensions


Extensions are additional properties used within PALS lattice files that are specific to a single machine or use case.
As such, additional dictionary entries can be added on the various levels of a lattice file.
Explicitly naming these extension dictionary key names is helpful to enable automatic finding user-typos or outdated keys in otherwise defaulted parameters, among other uses.

This section is meant to contain brief documentation on extensions to aid in identifying 
extension usage in lattice files and for directing to detailed extension documentation 
(which will live outside of the PALS documentation).
To get extension information into this section, a [pull request](#s:contribute) needs to be
created.

Syntax for including extension data in a lattice file is in the [Extension Syntax](#s:extension-syntax)
section.
